1. Select Bids archive

Please select a BIDS/.sef archive to import.

The files should be organized like this:

import.zip
- sub-[SubjectName]
- - ses-[examination-comment]
- - - [anat/func/dwi/eeg/ieeg/pet/..]
- - - - [list of images/nifti]

The subject will be automaticaly created, based on the subject name.

The examination will be automatically created based on examination comment and file creation date.

The data type folder (anat, func, etc) will define the type of dataset created in Shanoir

Data files will be split by name (there can be two files like nifti/json if the name is the same)
